Google Sheets Setup

Connect your application to Google Sheets database

Step 1: Create Sheets Structure

Your Google Sheet should have these sheets with the following structure:

Sheet 1: Users

Name | Email | Password | Role | Phone | SupervisorID | Status | CreatedAt

Sheet 2: Rooms

RoomNumber | Type | Price | Amenities | Status | CreatedAt

Sheet 3: Bookings

GuestName | GuestPhone | RoomNumber | CheckInDate | CheckOutDate | Guests | Amount | Status | CreatedBy | CreatedAt

Sheet 4: Payments

Date | GuestName | Amount | Method | Status | ProcessedBy | BookingID | CreatedAt

Sheet 5: Activities

Timestamp | UserName | Action | Description | UserID

Step 2: Get Google Sheets API Credentials

Follow these steps:

  1. Go to Google Cloud Console
  2. Create a new project
  3. Enable Google Sheets API
  4. Create OAuth 2.0 credentials (Desktop Application)
  5. Download the JSON file
  6. Copy your Spreadsheet ID from the URL

Step 3: Enter Your Credentials
